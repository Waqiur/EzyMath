Integration Basics

  1. \[\int x^n \,dx = \frac{x^{(n+1)}}{n+1}+c\]

  2. \[\int\frac{1}{x}\,dx = \ln\,|x|+c\]

  3. \[\int e^x\,dx = e^x+c\]

  4. \[\int a^x\,dx = \frac{a^x}{\ln\, a}+c\]

  5. \[\int \ln\, x\,dx = x\ln\, x-x+c\]

  6. \[\int \sin\, x\,dx = -\cos\, x+c\]

  7. \[\int \cos\, x\,dx = \sin\, x+c\]

  8. \[\int \sec^2\, x\,dx = \tan\, x+c\]

  9. \[\int \csc^2\, x\,dx = -\cot\, x+c\]

  10. \[\int \sec\, x\, \tan\, x\,dx = \sec\, x+c\]

  11. \[\int \csc\, x\, \cot\, x\,dx = -\csc\, x+c\]

  12. \[\int \tan\, x\,dx = \ln\,|\sec\, x|+c\]

  13. \[\int \cot\, x\,dx = \ln\,|\sin\, x|=-\ln\,|\csc\, x|+c\]

  14. \[\int \sec\, x\,dx = \ln\,|\tan\, x\,+\,\sec\, x|=-\ln\,|\sec\, x\,-\,\tan\, x|+c\]

  15. \[\int \csc\, x\,dx = \ln\,|\csc\, x\,-\,\cot\, x|=\ln\,|\tan\,\frac{x}{2}|+c\]

  16. \[\int \frac{1}{x^2+a^2}\,dx = \frac{1}{a}\tan^{-1}\left(\frac{x}{a}\right)+c\]

  17. \[\int \frac{1}{x^2-a^2}\,dx = \frac{1}{2a}\ln\,|\frac{x-a}{x+a}|+c\]

  18. \[\int \frac{1}{a^2-x^2}\,dx = \frac{1}{2a}\ln\,|\frac{x+a}{x-a}|+c\]

  19. \[\int \frac{1}{\sqrt{a^2-x^2}}\,dx = \sin^{-1}\left(\frac{x}{a}\right)+c\]

  20. \[\int \frac{1}{\sqrt{a^2+x^2}}\,dx = \ln\,|x+\sqrt{x^2+a^2}|+c\]

  21. \[\int \frac{1}{\sqrt{x^2-a^2}}\,dx = \ln\,|x+\sqrt{x^2-a^2}|+c\]

  22. \[\int \sqrt{a^2-x^2}\,dx = \frac{x}{2}\sqrt{a^2-x^2}+\frac{a^2}{2}\sin^{-1}\left(\frac{x}{a}\right)+c\]

  23. \[\int \sqrt{x^2-a^2}\,dx = \frac{x}{2}\sqrt{x^2-a^2}-\frac{a^2}{2}\ln\,|x+\sqrt{x^2-a^2}|+c\]

  24. \[\int \sqrt{x^2+a^2}\,dx = \frac{x}{2}\sqrt{x^2+a^2}+\frac{a^2}{2}\ln\,|x+\sqrt{x^2+a^2}|+c\]

  25. \[\int \frac{1}{|x|\sqrt{x^2-a^2}}\,dx = \frac{1}{a}\sec^{-1}\left(\frac{x}{a}\right)+c\]

  26. \[\int f(ax+b) \,dx = \frac{1}{a}\,f(ax+b)\,+c\]

  27. \[\int \frac{f'(x)}{f(x)} \,dx = \ln|f(x)|\,+c\]

  28. \[\int \frac{f'(x)}{\sqrt{f(x)}} \,dx = 2\sqrt{f(x)}\,+c\]

  29. \[\int (f(x))^{n}\,f'(x)\,dx = \frac{(f(x))^{(n+1)}}{n+1}+c\]

  30. \[\int e^{x}\,(f(x)\,+\,f'(x)\,)\,dx = e^{x}\,f(x)\,+c\]

  31. \[\int e^{ax}\sin bx \,dx = \frac{e^{ax}}{a^2+b^2}(a\sin bx-b\cos bx)+c\]

  32. \[\int e^{ax}\cos bx \,dx = \frac{e^{ax}}{a^2+b^2}(a\cos bx+b\sin bx)+c\]
