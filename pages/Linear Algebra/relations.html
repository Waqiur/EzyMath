Types of Relations

  1. Empty Relation

    An empty relation (or void relation) is one in which there is no relation between any elements of a set.

    R = φ ⊂ A × A

  2. Universal Relation

    A universal (or full relation) is a type of relation in which every element of a set is related to each other.

    R = A × A

  3. Identity Relation

    In an identity relation, every element of a set is related to itself only.

    I = {(a, a) : a ∈ A}

  4. Inverse Relation

    Inverse relation is seen when a set has elements which are inverse pairs of another set.

    R-1 = {(b, a) : (a, b) ∈ R}

  5. Reflexive Relation

    In a reflexive relation, every element maps to itself.

    (a, a) ∈ R

  6. Symmetric Relation

    In a symmetric relation, if a = b is true then b = a is also true.

    aRb ⇒ bRa, ∀ a, b ∈ A

  7. Transitive Relation

    For transitive relation, if (x, y) ∈ R, (y, z) ∈ R, then (x, z) ∈ R.

    aRb and bRc ⇒ aRc, ∀ a, b, c ∈ A

  8. Equivalence Relation

    If a relation is reflexive, symmetric, and transitive at the same time it is known as an equivalence relation.
